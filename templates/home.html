Weather Data API

TC:Celcius degrees ; TF:Fahrenheit degrees

Get temperature from a specific date

URL format: http://127.0.0.1:5000/api/v1/station/yyyy/mm/dd

URL example: http://127.0.0.1:5000/api/v1/10/1997/08/24

Get temperature from a month range

URL format: http://127.0.0.1:5000/api/v1/station/yyyy/mm

URL example: http://127.0.0.1:5000/api/v1/10/1997/08

Get temperature from a year range

URL format: http://127.0.0.1:5000/api/v1/station/yyyy

URL example: http://127.0.0.1:5000/api/v1/10/1997

In this case you can access data from decades, centuries, and millennia, providing less than 4 digits to the year argument.

i.e.: http://127.0.0.1:5000/api/v1/10/199 to get temperature data for the whole 90's decade.

Get all temperature data

URL format: http://127.0.0.1:5000/api/v1/station

URL example: http://127.0.0.1:5000/api/v1/10
